## 🚀 **첫 실행 체크리스트**

### **1단계: 코드 pull 받기**
```bash
cd h:\SPC_양산일정스케쥴러\ 프로젝트\manufacturing-scheduler-main

git fetch origin
git checkout donggun
git pull origin donggun
```

### **2단계: 환경변수 확인**
```bash
# .env 파일 확인 (루트에 있어야 함)
cat .env

# 반드시 확인할 값:
# - POSTGRES_PASSWORD
# - SECRET_KEY
# - OPENAI_API_KEY ← 반드시 설정되어야 함!
# - R2 자격증명 (모두 입력됨)
```

⚠️ **주의:** `.env` 파일은 git에 올라가지 않음!
- 필요한 경우 이전 PC에서 복사
- 또는 새로 설정

### **3단계: Docker 시작**
```bash
docker compose down -v          # 기존 상태 초기화
docker compose up -d            # 모든 서비스 시작

# 대기 (30초 정도)
docker compose ps               # 모두 Up 상태 확인

# 로그 확인
docker compose logs backend --tail 50
```

### **4단계: 준비 완료 확인**
```bash
# 프론트엔드 접속 (브라우저)
http://localhost

# 백엔드 API 확인
curl http://localhost/api/health

# DB 확인
docker compose exec -T db psql -U admin -d manufacturing_db -c "SELECT COUNT(*) FROM documents;"

# 응답이 숫자로 나오면 정상
```

```sql
-- 임베딩된 청크 샘플링
SELECT 
    dc.chunk_id,
    dc.chunk_index,
    SUBSTRING(dc.content, 1, 100) as content_preview,
    d.file_name
FROM document_chunks dc
JOIN documents d ON dc.file_id = d.file_id
WHERE d.file_path LIKE '%safety_manage%'
LIMIT 20;
```
